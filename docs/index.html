Mogy

A keyboard-driven MongoDB query UI with first-class Vim support

Vim-first

Full vim mode in editor. jk to exit insert mode, :w to save.

Keyboard-driven

Navigate everything with j/k/h/l, g/G, and leader key.

Query Execution

Run .find(), .aggregate(), .updateMany(), .count() and more.

Session Restore

Remembers last connection, database, collection, and content.
